# Installing Node.js for the Frontend

You need to install Node.js to run the frontend. Node.js includes npm.

## Quick Install

1. **Download Node.js:**
   - Go to: https://nodejs.org/
   - Download the **LTS version** (recommended)
   - Run the installer

2. **During installation:**
   - Check the box that says "Add to PATH" (usually checked by default)
   - Complete the installation

3. **Verify installation:**
   - Close and reopen your terminal
   - Run: `node --version`
   - Run: `npm --version`
   - Both should show version numbers

4. **Then continue with frontend setup:**
   ```bash
   cd C:\Users\rahul\Repos\llm-battle-arena\frontend
   npm install
   npm run dev
   ```
